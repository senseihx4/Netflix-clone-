# 🎬 Netflix Clone

A pixel-perfect front-end clone of the Netflix landing page, built with pure **HTML and CSS** — no frameworks, no libraries. Every section, layout, and style was hand-coded from scratch to replicate Netflix's UI as closely as possible.

---

## ✨ Features 

- 🎨 Pixel-accurate Netflix landing page recreation
- 📱 Fully responsive — works on mobile, tablet, and desktop        
- 🦸 Hero section with headline, subtitle, and email CTA
- 🃏 Feature cards (TV, download, kids profile, etc.)
- ❓ FAQ accordion section
- 🔤 Multi-column footer with links
- 🚫 Zero dependencies — pure HTML & CSS only

---

## 🛠 Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Semantic page structure |
| CSS3 | Flexbox, Grid, animations, media queries |

---

## 🚀 Getting Started

No setup, no install — just open in a browser.

```bash
# Clone the repo
git clone https://github.com/senseihx4/Netflix-clone-.git
cd Netflix-clone-

# Open directly in browser
open index.html        # Mac
start index.html       # Windows
xdg-open index.html   # Linux
```

Or download the ZIP and double-click `index.html`.
